# Meej

Meej is a headless client for Mixy written in Go.
It runs on both Windows and Linux and is configured using a YAML file.
Meej allows seamless integration and control of Mixy devices without a graphical interface.

Download latest build in [Releases](https://github.com/MixyLabs/meej/releases/latest)

## References

Meej is heavily based on [deej](https://github.com/omriharel/deej).
